# Quiz Application

## How to Run

1. Make sure you have Node.js installed.

2. Install dependencies:

```
npm install
```

3. Start the server:

```
npm start
```

4. Open your browser and go to:

```
http://localhost:3000
```

5. Enjoy the quiz!

## How to Add Questions

- Edit the `questions.json` file.
- Add questions with options and correct answer index.

That's it!
